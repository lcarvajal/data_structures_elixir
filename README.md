# DsEx

Practicing data structures with Elixir.

Documentation for each data structure is under `doc/index.html`.

## Roadmap

- [x] Stack
- [x] Queue
- [x] Linked List
- [ ] Binary Tree
- [ ] Binary Search Tree
- [ ] Hash Map (Dictionary)
- [ ] Set
- [ ] Graph
- [ ] Heap (Priority Queue)
- [ ] Trie (Prefix Tree)

## Elixir commands

- `mix deps.get` - get dependencies
- `mix docs` - generate docs
- `mix test` - run tests
